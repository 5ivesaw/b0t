# Security Policy

SawBotV1 is restricted to single-player, LAN, owned private servers, and purpose-built local arenas.

Do not submit or request anti-cheat bypasses, authentication bypasses, credential collection, token access, packet manipulation for advantage, altered reach, impossible placement, public-server automation, or concealed surveillance functionality.

For a security-sensitive defect, avoid publishing secrets, session tokens, private server addresses, or personal information in a public issue. Provide a minimal sanitized reproduction and identify the affected commit and release.
